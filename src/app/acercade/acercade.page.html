Acerca de Atrás RegistrAPP Duoc Alumnos, es una aplicación que forma parte del proyecto Vivo duoc, una iniciativa que tiene por objetivo mejorar la experiencia de los alumnos tanto profesores durante la pandemia haciendo esta una experiencia mas amigable.
Esta aplicación movil fue creada para que puedes tener toda la información que necesitas para estar al dia con las asistencias a clases desde cualquier lugar y cuando quieras.
